# MikroTik Controller

Spec-driven project for a SaaS multi-tenant MikroTik controller platform.

## Structure

- `docs/specs/` → source-of-truth specifications
- `ai/` → prompts for AI coding tools
- `backend/` → backend code
- `frontend/` → frontend code

## Recommended workflow

1. Open the repository in your AI coding environment.
2. Ask the model to read all files in `docs/specs`.
3. Use the specs as the source of truth.
4. Generate architecture, then modules one by one.
